Политика конфиденциальности приложения «Изучаем итальянский»
Дата: 11.01.2026
1. Какие данные мы собираем
Приложение не собирает и не запрашивает персональные данные пользователя (имя, адрес, телефон, электронную почту и т.п.).
Приложение не требует регистрации.
2. Как используются данные
Так как персональные данные не собираются, они не используются и не передаются.
3. Передача третьим лицам
Приложение не передаёт персональные данные третьим лицам.
4. Хранение данных на устройстве
Прогресс обучения и настройки могут храниться локально на устройстве пользователя и не отправляются на внешние серверы.
5. Связь с нами
По вопросам конфиденциальности напишите: ВСТАВЬ_СВОЮ_ПОЧТУ
6. Изменения политики
Мы можем обновлять эту политику. Актуальная версия всегда доступна на этой странице.
